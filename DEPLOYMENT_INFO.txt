DEPLOYMENT INSTRUCTIONS
=================================================================

1. Extract the ZIP file to your web server's public directory

2. Ensure your web server is configured to:
   - Serve .js files with correct MIME type (text/javascript)
   - Allow .htaccess files (if using Apache)

3. Access the application:
   - Main Hub: http://yourserver.com/
   - Direct to PM Tool: http://yourserver.com/pmtool.html
   - Direct to Action Items: http://yourserver.com/actions.html

4. Browser Requirements:
   - Modern browser with JavaScript enabled
   - React 18 compatible
   - Internet connection for CDN resources (React, SheetJS)
